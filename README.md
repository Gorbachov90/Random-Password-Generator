# Автор: Горбачев Роман

## Описание
Это приложение для генерации случайных паролей с настройками длины и символов. Можно сохранять историю созданных паролей и загружать её. Подходит для быстрого создания сложных паролей.

## Использование
1. Запустите программу.
2. Настройте длину пароля с помощью ползунка.
3. Выберите используемые типы символов (цифры, буквы, спецсимволы).
4. Нажмите кнопку "Генерировать пароль".
5. Сгенерированный пароль появится в поле, а автоматически сохранится в историю.
6. Используйте кнопки для сохранения и загрузки истории.

## Требования
- Python 3.x
- Tkinter (обычно входит в стандартную библиотеку)
